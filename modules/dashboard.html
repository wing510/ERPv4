Dashboard 儀表板

重點在「今天要處理什麼」與「風險提醒」，而非單純筆數統計。

首次使用？ 建議操作順序：
1. 先至「產品」「供應商」建立主檔 → 2. 再建「採購單」或「進口報單」 → 3. 到「Goods Receipt 收貨入庫」選來源、填收貨數量產生批次 → 4. 到「Lots 批次QA管理」做 QA 放行。
待辦與風險（批次／效期）

無法讀取庫存異動彙總（available map）。目前可用量將顯示為「—/--」，請稍後重試或確認後端是否可連線。

偵測到有 Lot 缺少 movement（庫存流水帳）紀錄：— 筆。這些 Lot 的可用量會顯示為「--」，且出貨/扣庫會被阻擋。
